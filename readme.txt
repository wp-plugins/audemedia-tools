== Description ==

Audemedia Tools is a set of plugins that extend Audemedia's WordPress Themes. Contains a set of Custom Post Types like Portfolio, Services, Testimonials and Slider and also shortcodes and widgets.

== Installation ==

= Minimum Requirements =

* WordPress 3.5 or greater
* PHP version 5.2.4 or greater
* MySQL version 5.0 or greater

= Automatic Installation =

1. Log in and navigate to Plugins &rarr; Add New.
2. Type "Audemedia Tools" into the Search input and click the "Search Plugins" button.
3. Locate the Audemedia Tools in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.

= Manual Installation =

1. Download the "Audemedia Tools" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Audemedia Tools" in the list and click the "Activate" link.

== Frequently Asked Questions ==

= What is this plugin and why do I need it? =

The Audemedia Tools provides extra functionality to Audemedia's WordPress theme. It includes several custom post types, shortcodes and widgets  (only for Audemedia WordPress Themes). This plugin is not requirement to use Audemedia WordPress themes however it extends the theme functionality.

= Can I use this plugin with other themes? =

The Audemedia Tools was developed to work only with Audemedia WordPress themes.
